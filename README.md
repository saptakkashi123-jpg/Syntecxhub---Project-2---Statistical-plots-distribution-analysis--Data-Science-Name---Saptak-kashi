📊 Project 2: Statistical Plots & Distribution Analysis
📌 Overview
This project focuses on Statistical Plots & Distribution Analysis using Python. The objective is to explore and understand data distributions through various visualization techniques, compare distributions across different groups, identify outliers, and analyze skewness and spread. The project demonstrates how statistical visualization can reveal meaningful insights from data for better decision-making.
🚀 Objectives
•	Visualize data distributions using Histograms, KDE plots, and Boxplots.
•	Compare distributions between Region A and Region B.
•	Detect outliers using the Interquartile Range (IQR) method.
•	Analyze skewness, spread, and variability of the data.
•	Export generated plots and an interpretation report.
